# 简介

这是yelp点评数据分析与推荐项目的前端仓库，完成了大数据分析及可视化，以及大数据应用开发

## 项目初始化

```sh
npm install
```

### 编译和热加载

```sh
npm run dev
```

### 编译打包

```sh
npm run build
```
